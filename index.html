olá, sou um estudante de ciência da computação e fiz este portifólio para mostrar alguns de meus projetos.

Plantae

O Plantae foi um projeto de TCC desenvolvido por mim e mais 4 amigos onde fizemos uma estufa com o intuito de automatizar o plantio domestico, utilizando de diversos sensores e um reservatório de água para proporcionar uma autonomia de alguns dias para a planta. O usuário poderia verificar status da mesma pelo site, com informações sobre os sensores.

OTSK

Only the Skates Know é um jogo desenvolvido por mim e mais 3 amigos sobre um skatista em uma jornada para conseguir uma parte de um disco de vinil, nele o jogador deve passar por uma parte plataformer e outra de combate com um boss ao final.
Desenvolvemos também um site para o jogo, contendo até uma lojinha virtual com alguns produtos.

Reprodutor de musica

Este reprodutor de música foi um projeto feito utilizando o Processing onde eu e minha equipe possuíamos o intuito de fazer um aplicativo semelhante ao Spotify, para que existisse um diferencial, colocamos gifs nos álbuns das músicas e um efeito de sound wave na parte de baixo da tela.

The Curse of the 4

The Curse of the 4 é um jogo que desenvolvi com uma equipe de mais 3 pessoas, queríamos fazer um jogo com câmera top-down onde o jogador deveria enfrentar diversos bosses(5 ao todo) controlando o mago Alatar e utilizando de skills como uma bola de fogo um escudo magico e um dash.
O projeto foi feito utilizando o Construct 2
